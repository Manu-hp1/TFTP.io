UDP-Based
TFTP Implementation

A custom implementation of the Trivial File Transfer Protocol (TFTP) using UDP sockets in C, supporting file upload and download operations between client and server systems.

Source Code Watch Demo

Core Functionalities

Client–Server Model

Implements file transfer using UDP socket communication.

PUT & GET Operations

Supports file upload and download requests.

Packet Handling

Implements RRQ, WRQ, DATA, ACK and ERROR packets.

Key Features

  • UDP socket programming
  • Client-server based file transfer
  • Support for file upload (PUT) and file download (GET)
  • Handles text (netascii) and binary (octet) transfer modes
  • Structured DATA and ACK packet handling
  • Simple and readable logging for packet flow and errors
  • Designed for Linux environments

Technology Stack
